Cat API

There is one endpoint /breed that can be accessed in three ways:

  • ./.netlify/functions/breed returns an array of {id, name}
  • ./.netlify/functions/breed/42 returns a breed object of id OR null if not found
  • ./.netlify/functions/breed?search=bobtail returns an array of breed objects or an empty array if not found

An example breed object:

{ name: "Aegean", country: "Greece", origin: "Natural", coat: "Semi-long", pattern: "Bicolored or tricolored", temperament: "Even-tempered" }
