Cat API

The Cat API accesses cat breeds

There is one endpoint /breed that can be accessed in three ways:

  • ./.netlify/functions/breed returns an array of {id, name} objects representing all breed names and their corresponding ids in the database
  • ./.netlify/functions/breed/{ id } returns a breed matching id or null if not found (e.g. 5cad85a5f720f54848771d87
  • ./.netlify/functions/breed?search=bobtail returns an array of breeds or an empty array if not found
